Playroll

Nebula VR

Nebula VR is a virtual reality IDE used for the visual scripting language Nebula. The IDE was developed in Unity for the HTC Vive and was developed for LMU's CMSI 401: Software Engineering Laboratory course. It takes the coordinates of game objects within the environment and translates that to the Nebula Javascript compiler, then sends the output back to the VR environment. This is communicated through the MVVM (model-view-viewmodel) architecture.

I headed the "view" aspect of this project, working solely on the Unity side. I designed the interactions between function/variable blocks (see presentation linked below) including spawning blocks, grabbing blocks, and linking blocks together. I wrote a large part of the Unity C# of this project and was happy to have such a new topic like virtual reality be my introduction to collaborative software engineering.

casper

"casper" is a scripting language designed for LMU's CMSI 488: Language Translation and Implementation (aka Compilers) course of Spring 2019. The language's is built using Ohm, compiled to Javascript, and tested using Mocha and ESLint. The Ohm grammar along with parser tests are available on the GitHub repo, and abstract syntax trees and semantic analysis is on the way.

"casper" is statically typed and includes partial type inference, parallel variable declaration and assignment, higher order functions, and optional parameters. It takes inspiration from Python and Typescript to create a clean-looking interview language to practice in.

Currently, I am the working project manager and am currently finalizing the grammar and developing the abstract syntax trees for the language.
