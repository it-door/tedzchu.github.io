casper

"casper" is a scripting language designed for LMU's CMSI 488: Language Translation and Implementation (aka Compilers) course of Spring 2019. This language is built using Ohm and tested using Javascript including Mocha and ESLint. The Ohm grammar along with parser tests are available on the GitHub repo, and semantic analysis is on the way.

"casper" is statically typed and includes partial type inference, parallel variable declaration and assignment, higher order functions, and optional parameters. It takes inspiration from Python and Typescript to create a clean-looking interview language to practice in.

"casper" is being worked on in collaboration with Alexia Filler, Ian Lizarda, Donovan Moini, and Serena Zafiris.
